How to Use This Visualizer
  1. Select a mode: Segmentation, Paging, or Virtual Memory.
  2. Use Load Example to start quickly or enter your own data.
  3. Click Run Simulation to generate animated output.
  4. In Virtual Memory, use step controls to play each memory access.
  5. Compare page faults/hits across algorithms for analysis.
